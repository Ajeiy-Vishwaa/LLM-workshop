# 🪔 SiddhaGPT – Siddha Medical Study Assistant

SiddhaGPT is an AI-powered Siddha Medical Study Assistant built using a Large Language Model (LLM) API.  
It answers Siddha-based theory questions and helps B.S.M.S students understand traditional Siddha concepts in a structured and exam-friendly format.

---

## 📌 Project Description

SiddhaGPT uses prompt engineering techniques to guide a powerful LLM to behave like a Siddha medical professor.  
The system focuses on delivering structured, student-friendly explanations strictly based on Siddha medicine principles.

---

## 🎯 Objectives

- Provide Siddha theory explanations
- Help students prepare for university examinations
- Explain concepts such as:
  - Mukkutram (Vatha, Pitha, Kabam)
  - 96 Thathuvam
  - Gunapadam
  - Siddha diagnostic methods
  - Disease classifications
- Deliver structured short notes for revision

---

## 🏗 System Architecture

User Question  
      ↓  
Custom Siddha Prompt  
      ↓  
LLM API (Gemini / OpenAI)  
      ↓  
Structured Siddha Answer  

---

## ⚙️ Technologies Used

- Python 3.x
- LLM API (Google Gemini or OpenAI GPT)
- Prompt Engineering
- Google Colab / VS Code
